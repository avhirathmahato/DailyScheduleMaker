# Daily Schedule Builder 🕒

A simple, interactive, single-page web application that helps users structure their day by answering questions using responsive range sliders.

## Features
- **Interactive Range Sliders:** Custom slider inputs representing minutes throughout the day.
- **Real-time Formatting:** Live AM/PM preview updates instantly as you drag.
- **Clean Output:** Generates a structured daily routine summary list at the end.
- **Mobile Friendly:** Fully responsive design built using HTML5, CSS3, and vanilla JavaScript.

## How to Run Locally
1. Clone the repository.
2. Open `index.html` directly in any standard modern browser.
